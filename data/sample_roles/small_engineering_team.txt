Small Engineering Team - Role Definitions
==========================================

Team Lead - Engineering
-----------------------
Oversees a small team of engineers working on product development. Combines hands-on technical work with people management.

Key responsibilities include:
- Technical leadership and code reviews
- Sprint planning and team coordination
- Mentoring junior team members
- Stakeholder communication
- Resolving technical blockers
- Contributing to architecture decisions

Skills: Technical expertise, leadership, agile methods, communication


Senior Engineer
---------------
Experienced individual contributor who takes ownership of complex features and provides technical guidance.

Key responsibilities include:
- Designing and implementing major features
- Writing technical documentation
- Conducting code reviews
- Troubleshooting complex issues
- Evaluating new technologies
- Supporting production systems

Skills: Deep technical expertise, problem-solving, documentation, system design


Engineer
--------
Core team member who delivers features and fixes bugs. Works collaboratively with the team.

Key responsibilities include:
- Implementing features per specifications
- Writing unit and integration tests
- Participating in code reviews
- Debugging and fixing issues
- Updating documentation
- Learning new technologies

Skills: Programming, testing, debugging, teamwork, continuous learning


Junior Engineer / Intern
------------------------
Entry-level position focused on learning and contributing to the team under guidance.

Key responsibilities include:
- Working on well-defined tasks
- Learning codebase and processes
- Asking questions and seeking feedback
- Participating in team meetings
- Documenting learnings
- Gradually taking on more complex work

Skills: Foundational programming, eagerness to learn, communication
